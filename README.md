# Grand Hotel 1908

Projeto front-end estático que apresenta a rede fictícia Grand Hotel, com foco em narrar experiências de hospedagem premium em diferentes destinos e oferecer um fluxo simples de reserva online.

## Tecnologias usadas
- HTML5 semântico para estruturar conteúdo e melhorar a acessibilidade
- CSS3 com variáveis customizadas, Flexbox e Grid para composições responsivas e consistência visual
- Boas práticas de UX com navegação fixa, chamadas para ação destacadas e formulários validados pelo navegador

## Estrutura de páginas
- `index.html`: página inicial com destaques da marca, catálogo dos hotéis e galeria de experiências
- `sobre.html`: narrativa da história, missão e diferenciais da rede Grand Hotel
- `pacotes.html`: listagem de pacotes temáticos com benefícios e valores indicativos
- `servicos.html`: vitrine detalhada de serviços (restaurante, piscina, academia, comodidades)
- `contato.html`: canal direto para dúvidas e mensagens gerais
- `reserva.html`: formulário acessível para solicitar reservas e informar preferências
- Demais páginas (`hoteldolomitas.html`, `hotelgramado.html`, `hotellosandes.html`) apresentam conteúdos específicos de cada unidade

## Organização de arquivos
- `assets/css/style.css`: folha de estilos única, com design system baseado em variáveis e componentes reutilizáveis
- `assets/img/`: imagens otimizadas por destino (Brasil, Chile e Itália) e ícones institucionais

## Como executar
1. Faça o clone ou download do repositório.
2. Abra o arquivo `index.html` no navegador de sua preferência.
3. Navegue pelas páginas por meio do menu fixo no topo.

Para editar estilos, ajuste `assets/css/style.css`. As páginas HTML se beneficiam da mesma folha de estilos, facilitando a manutenção visual do site.
